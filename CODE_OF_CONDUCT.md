# Code of Conduct

## Expected Behavior

- Be respectful, specific, and constructive.
- Focus technical discussion on observable behavior, tradeoffs, and evidence.
- Respect privacy, security boundaries, and coordinated vulnerability disclosure.
- Do not use discriminatory, harassing, threatening, or abusive language.

## Unacceptable Behavior

Harassment, personal attacks, publication of private information, deliberate
security abuse, and disruption of project collaboration are not accepted.

## Reporting

Report conduct concerns privately to the repository maintainer. Security
vulnerabilities must follow [SECURITY.md](SECURITY.md).

Maintainers may remove content or restrict participation when necessary to
protect contributors and the project.
